# AtalaIA Licit@ — Demo Interactiva

Demo interactiva (2–3 min) de la plataforma **AtalaIA Licit@**, "la IA del licitador". Recorre guiadamente los dos módulos de la plataforma:

- **Módulo Directivo** — visión ejecutiva del pipeline de licitaciones: KPIs, rendimiento por administración y decisiones GO/NO-GO sobre nuevas convocatorias.
- **Módulo Técnico** — predicción de puntuación técnica por criterio, alertas de riesgo basadas en el histórico del evaluador, y plan de acción para recuperar puntos antes de presentar la oferta.

La demo es un único archivo HTML autocontenido (sin dependencias externas, sin build, sin backend) — funciona abriéndolo directamente en el navegador.

## Publicar con GitHub Pages

1. Sube este repositorio a GitHub (este `README.md` + `index.html` en la raíz).
2. En el repo: **Settings → Pages → Source**, selecciona la rama `main` y la carpeta `/ (root)`.
3. GitHub publicará la demo en `https://<usuario>.github.io/<repo>/` en un par de minutos.

## Uso local

Abre `index.html` directamente en cualquier navegador — no requiere servidor.

## Navegación de la demo

- Botones **Siguiente / Volver** o flechas del teclado (← →).
- Puntos de progreso en portada y cierre.
- Interacciones reales sobre tarjetas y enlaces del propio producto (p. ej. "Ver informe técnico →").
